# 04 — Opérations

Ce dossier rassemble les procédures qui font fonctionner l'entreprise et la plateforme : installation, exploitation, automatisations et checklists.

## Structure actuelle

- `docs/` : guides techniques existants, setup PostgreSQL, quickstart et statut projet.

## Sous-dossiers recommandés

- `process/` : modes opératoires récurrents.
- `automatisations/` : scripts et scénarios d'automatisation.
- `administratif/` : documents administratifs.
- `partenariats/` : ressources partenaires.
- `finances/` : suivi financier.
- `checklists/` : listes de contrôle avant publication, import ou livraison.
