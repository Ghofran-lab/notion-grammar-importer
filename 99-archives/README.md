# 99 — Archives

Ce dossier est réservé aux contenus obsolètes, anciens projets, documents remplacés ou historiques à conserver sans encombrer les piliers actifs.

Avant de supprimer un élément important, le déplacer ici avec une note courte expliquant :

- ce que c'était ;
- pourquoi c'est archivé ;
- la date d'archivage ;
- le lien vers le nouveau document si applicable.
